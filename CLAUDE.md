# Claude Instructions — DGX Spark Bundle

## Project overview

Static GitHub Pages site hosting open technical books on AI infrastructure. Built with plain HTML/CSS — no build pipeline, no framework.

## Structure

- `index.html` — root landing page (books catalog)
- `books/from-box-to-cluster/index.html` — single-page ebook
- `books/from-box-to-cluster/styles.css` — book stylesheet
- `books/from-box-to-cluster/dist/` — generated PDF/HTML artifacts
- `.github/workflows/pages.yml` — GitHub Pages CI

## Conventions

- **No build step** — all HTML is hand-authored; do not introduce bundlers or frameworks
- **Single-page ebook** — the book lives in one HTML file with in-page anchor navigation
- **SEO/AEO** — every page carries full JSON-LD structured data and Google Analytics `G-KPMCQ8HS3P`
- **Google Analytics** — tag is placed immediately after `<head>` open tag on every HTML page
- **Canonical URLs** — always `https://mohnishbasha.github.io/dgx-spark-bundle/...`

## Do not

- Introduce JavaScript frameworks or bundlers
- Remove or alter Google Analytics snippet
- Remove structured data (JSON-LD) blocks
- Modify `dist/` artifacts manually — they are generated by `build.sh`
